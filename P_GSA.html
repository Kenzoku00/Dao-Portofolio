Project Details

Details

Guess Strange Animal is a fun guessing game created during the Global Game Jam Malang 2024 with the theme "Make Me Laugh." Players must identify strange animals presented at each level using hints that provided.

As the game designer, I was responsible for searching strange animals, along with designing the hints, blending humor and clever clues to guide players without making the game too easy. The game is filled with humor and surprises, making it a delightful challenge for players who enjoy quirky puzzles and unexpected laughs.
